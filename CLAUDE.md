# 주사위 스택 (Dice Stack) — 개발 가이드

## 프로젝트 개요
웹 브라우저에서 바로 플레이 가능한 미니게임.
주사위를 모양 블록에 채워 보드에 떨어뜨리고, 같은 숫자 3칸을 맞춰 제거하는 퍼즐 게임.

---

## 기술 스택
- **Phaser.js 3** (CDN: https://cdn.jsdelivr.net/npm/phaser@3/dist/phaser.min.js)
- 단일 파일: `index.html` (HTML + JS 인라인)
- 외부 라이브러리: Phaser.js CDN만 사용
- GitHub Pages 배포 기준

---

## 씬(Scene) 구성

| 씬 이름 | 역할 |
|---------|------|
| `BootScene` | 로딩, 에셋 준비 |
| `TitleScene` | 타이틀 화면, 시작 버튼 |
| `GameScene` | 메인 게임 로직 |
| `GameOverScene` | 게임 오버, 최종 점수 / 라운드, 다시 시작 |

---

## 게임 구조

### 그리드
- **5열 x 16행**
- 테트리스처럼 아래에서 위로 쌓임
- 최상단(0행)에 주사위가 닿으면 게임 오버
- 각 셀 크기: 64x64px

### 주사위
- 눈: 1~6
- UI 표시: 주사위 눈 이모지 (⚀⚁⚂⚃⚄⚅) 또는 직접 그린 스프라이트
- **방해 주사위**: ✕ 표시, 붉은색 배경 (제거 조건 동일하게 적용)

---

## 플레이어 턴 흐름

### 1단계: 주사위 굴림
1. 한 번에 주사위 **10개** 굴림 (랜덤 1~6)

### 2단계: 모양 채우기 & 배치 (반복)
1. **시스템이 랜덤 모양** 생성 (1칸 / 2칸 / 3칸, 회전 가능)
2. 플레이어가 굴린 주사위 풀에서 **키보드로 주사위를 선택**해 모양의 각 슬롯에 채움
3. 모양의 슬롯이 모두 채워지면 **보드에 배치** (플레이어가 열 위치 선택 후 드롭)
4. 배치 후 보드 정산 (3칸 제거 → 중력 → 연쇄 체크)
5. Next 큐의 다음 모양이 현재 모양이 됨 → 2단계 반복
6. 주사위가 부족하면 **자동 재굴림**으로 풀을 다시 채움

---

## 모양 시스템

### 모양 종류 (예시)
- 1칸: 단일 주사위
- 2칸: 가로 2칸, 세로 2칸
- 3칸: 가로 3칸, 세로 3칸, L자형, 대각형 등

### 모양 조작 (키보드)
| 키 | 동작 |
|----|------|
| `←` `→` | 보드에서 모양 좌우 이동 |
| `R` | 모양 회전 |
| `Space` / `↓` | 모양 드롭 (배치 확정) |
| `1`~`0` | 주사위 풀에서 해당 번호 주사위 선택해 슬롯에 채움 |

### Next 큐
- 다음 3개 모양을 미리보기로 표시
- 현재 모양 배치 완료 시 Next 큐에서 자동으로 다음 모양 등장

---

## 제거 조건 (재귀적)
- **가로 / 세로 / 대각선 3칸 연달아 같은 숫자** → 해당 주사위 제거 + 점수
- 방해 주사위도 3칸 조건 충족 시 동일하게 제거
- 제거 후 중력 적용 → 다시 3칸 체크 (연쇄 가능)

---

## 점수 시스템
- 3칸 제거: 100점
- 연쇄 시: 체인 수 × 100점 추가 보너스

---

## 방해 주사위
- **모양 4개 배치마다** 랜덤 빈 위치에 3개 생성
- ✕ 표시, 붉은색 배경
- 3칸 조건 동일하게 제거 가능 (방해 주사위끼리도 가능)

---

## 중력 시스템
- **항상 작동**: 주사위는 언제나 자기 열의 맨 아래 빈 칸으로 떨어짐
- 모양을 배치할 때도 각 주사위가 해당 열의 바닥으로 **독립적으로** 낙하 (모양이 통째로 떨어지는 게 아님)
- 3칸 제거 후에도 위에 남은 주사위들이 아래로 내려옴
- 방해 주사위도 동일하게 중력 적용

---

## Phaser.js 애니메이션 요구사항

| 이벤트 | 애니메이션 |
|--------|-----------|
| 모양 드롭 | Tween으로 위에서 아래로 떨어지는 효과 |
| 3칸 제거 | 스케일 업 후 페이드아웃 (파티클 추가 권장) |
| 연쇄 발생 | 화면 흔들림 (Camera shake) |
| 방해 주사위 등장 | 랜덤 위치에 팝업 애니메이션 |
| 게임 오버 | 화면 페이드아웃 후 씬 전환 |
| 주사위 굴리기 | 숫자 빠르게 변하는 롤링 효과 |
| 모양 이동 | 좌우 슬라이드 효과 |

## 코드 작성 규칙
- 각 씬은 클래스로 분리 (`class TitleScene extends Phaser.Scene`)
- 게임 상태는 `GameScene` 내부에서 관리
- 애니메이션은 Phaser Tween / Particle 시스템 활용
- 함수명은 영어, 주석은 한국어
- 가독성 우선, 과도한 최적화 지양
